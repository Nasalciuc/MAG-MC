# MAG-MC

Calculator web pentru **Matricea Duratelor** și **Grafic-Rețea (CPM)** — aplicație educațională pentru cursul UTM *Management în Construcții*.

**Repository:** [github.com/Nasalciuc/MAG-MC](https://github.com/Nasalciuc/MAG-MC)

## Proiect

Codul sursă se află în folderul [`mag-calculator/`](mag-calculator/). Aplicația principală este versiunea **React + TypeScript + Vite** (v3.0).

## Start rapid

```bash
cd mag-calculator
npm install
npm run dev
```

Deschide în browser adresa afișată de Vite (de obicei `http://localhost:5173`).

## Comenzi utile

| Comandă | Descriere |
|---------|-----------|
| `npm run dev` | Server de dezvoltare |
| `npm run build` | Build producție (`dist/`) |
| `npm run preview` | Previzualizare build local |
| `npm run test` | Teste Vitest (motor CPM) |
